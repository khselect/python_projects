# 원본 데이터 배치 안내

이 폴더에 아래 두 개의 원본 엑셀 파일을 넣어주세요 (파일명은 자유, ETL 실행 시 경로로 지정).

1. **초미세먼지 월보** — 시트 구성: `일평균`, `01`~`31`(일자별), `공단양식`(장문형, 실제 적재에 사용)
   - `공단양식` 시트 컬럼: 역명 / 지점코드 / 측정시간(YYYYMMDDHH) / 항목코드 / 시간 / 측정값 ...
2. **역·시간대별 통행량** — 시트명 `1호선 ` (뒤에 공백 포함), 컬럼: 역명 / 구분(승차·하차) / 계 / 05시~24시이후 20개 시간대

이 폴더는 `.gitignore`에 의해 원본 엑셀이 저장소에 커밋되지 않도록 설정되어 있습니다(개인/공공데이터이므로 git에는 올리지 않는 것을 권장).
